==============================================================================
Getting Started
==============================================================================

Prequisites

::

    1. install intel opencl sdk
       - ref: https://gist.github.com/rmcgibbo/6314452
       - path: /opt/intel

    2. install nvidia cuda
       - path: /usr/local/cuda-xx

    3. check *.icd
       - path: /etc/OpenCL/vendors/

    4. run `make platforms`

Compiling

::

    $ make compile
